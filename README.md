# My personal website at [amirsolo.com](https://amirsolo.com)

## Build with:

- ### [Next.js](https://nextjs.org)
- ### [Tailwind CSS ](https://tailwindcss.com)

---

Available scripts:

```shell
# runs the app in development mode
$ npm run dev
```

```shell
# makes production build
$ npm run build
```

```shell
# runs the app in production mode
$ npm start
```
